# 🏠 Smart Home Automation System

## 📌 Overview
This project is an IoT-based smart home automation system using Raspberry Pi and ESP8266. It allows users to control electrical appliances remotely using a web interface over WiFi.

---

## ⚙️ Hardware Used
- Raspberry Pi (3/4 recommended)
- ESP8266 WiFi module
- Relay module (for controlling appliances)
- Jumper wires
- Breadboard

---

## 💻 Software Used
- Python (Raspberry Pi side)
- Arduino IDE (ESP8266 programming)
- Flask (web interface)
- MQTT or HTTP communication

---

## 🚀 Features
- Control appliances remotely via WiFi
- Web-based control dashboard
- Real-time ON/OFF switching
- Scalable IoT architecture
- Works over local network

---

## 🛠️ How It Works
1. ESP8266 connects to WiFi
2. Raspberry Pi runs a server (Flask/MQTT broker)
3. Commands are sent from web dashboard
4. ESP8266 receives commands and controls relay
5. Appliances are switched ON/OFF
